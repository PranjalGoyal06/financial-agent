# SCALE – Machine Learning Club

## Problem Statement 1: Build an AI Agent for Finance

**Stakeholders:** Kalp Shah, Jils Shah

This PS is inspired from the Pathway Problem Statement of Inter-IIT Tech Meet 14.0.

Build an AI-powered Financial Agent using LangChain that can optimize financial decisions and maximize savings or profits.

The system should integrate:
- LLM reasoning
- External financial APIs/tools
- Memory
- Agent-based workflows

to provide accurate, contextual, and actionable financial assistance in real time.

## Common Examples

- AI stock portfolio advisor with risk analysis and market sentiment tracking
- Credit score improvement advisor with personalized financial recommendations
- Financial news summarization agent with impact analysis on markets and portfolios

This Problem Statement has been deliberately kept open and rewards novel ideas.
